Z370-G Hackintosh package

Step1: Prepare USB:
    Use unibeast + efi folder 
    
Step2: BIOS setting:
    needs update

Step3: Install
    Boot in and install

    Install Clover
    Replace local efi folder
    
Step5: run post_install
    post install will install the following:
        1. homebrew
        2. webdriver
        3. python + pip
        4. python + pip3
